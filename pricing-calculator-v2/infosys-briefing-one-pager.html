Pricing Model — At a Glance

A unified, deterministic, spend-anchored framework for enterprise deals from $100M to $10B+ SUM.

TL;DR. Aerchain pricing is a three-layer framework: a spend-tier anchored subscription (45% Platform + 55% Workflow), a soft multiplier layer for user and transaction volume, and a smooth volume-discount power curve (up to 70% off at $3B). Implementation services scale with deal size via a square-root curve. Every component is pure deterministic code — no AI, no hidden logic — and fully configurable via JSON without code changes. A 10% manual discount applies proportionally across subscription, services, and gain share.

Calculation Chain

spend × product × entity → Baseline Total × (1 − volumeDiscount) → Net Baseline × 45% platform × userMult → Platform Access × 55% workflow × volumeMult → Workflow Fees + integrations + impl × spendScale → Gross Y1 × (1 − manualDiscount) → Y1 Total Commitment

The Three Multiplier Layers

1 · User Multiplier

Applies to: Platform Access (45%)

Range: [0.85x, 1.15x] clamped

clamp(0.85, 1.15, (users/default)^0.3)

Soft ±15% scaling based on power/light user count vs. tier benchmark. Power users count 5× more.

2 · Volume Multiplier

Applies to: Workflow Fees (55%)

Range: [0.85x, 2.00x] clamped

clamp(0.85, 2.00, (actual/expected)^0.3)

Asymmetric scaling based on actual monthly transactions vs. preset expectation. Heavy users pay up to 2×.

3 · Impl Spend Scale

Applies to: Services (above $750M)

Range: [1.0x, 3.0x] clamped

clamp(1.0, 3.0, (spend/$750M)^0.5)

Square-root curve reflects diminishing-returns reality — a $3B rollout is ~2x the effort of a $750M one.

Scope Presets & Complexity-Weighted Channels

Deals only activate the channels the client is buying. The workflow budget is distributed across active channels using complexity-weighted normalization: weight[ch] = complexity[ch] / Σ complexity[active]. No dilution across channels the client isn't using; per-transaction prices are back-derived to always sum to the workflow budget target.

Pricing Advantage

Aerchain runs 30–55% below legacy S2P rates at equivalent scope while delivering agentic sourcing, autonomous negotiation, and AI-driven strategic sourcing. The volume discount curve is built into the framework — no one-off exceptions. Large deals get aggressive pricing by design.

Key Defensibility Points

Deterministic

All calculations are pure code. Same inputs always produce same outputs. AI is used for context parsing and narrative only — never for the math itself.

Configurable

Every parameter — spend tiers, multipliers, discount curves, channel weights, rate cards — lives in JSON. Sales leadership tunes directly, no developer involvement.

Auditable

Every dollar traces back to a specific component. Clients can reproduce any calculation in Excel given the same inputs. Zero hidden logic.
